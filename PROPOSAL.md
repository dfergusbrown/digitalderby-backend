## User Stories

### MVP
- As a user, I want to be able to create a user account (C): so that I can sign up for the website and start betting.
- As a user, I want to be able to login into the user account (R): to access my account and place bets securely.
- As a user, I want to be able to see the information regarding my account (R): so that I can keep track of my profile details and betting history.
- As a user, I want to be able to update the user account (U): to modify my account information such as username or password.
- As a user, I want to be able to delete the user account (D): if I choose to discontinue using the platform.
- As a user, I want to be able to navigate through the website: to easily find the information I need and place bets.
- As a user, I want to be able to see a list of all the race horses available: so that I can make informed betting decisions based on horse details and performance.
- As a user, I want to be able to place a bet on the selected horse: to participate in the races and potentially win.
- As a user, I want to be able to see a real-time representation of the horses racing: to follow the progress of the races as they happen.
- As a user, I want to be able to see the static race result image: to know the final positions of the horses after each race.
- As a user, I want to be able to see the winning/losing breakdown: to understand the outcomes of each race and potential returns.
- As a user, I want to be able to see a timer until the next race/close of betting: to plan my betting activities and know when the next race starts or when betting closes.
- As a user, I want to be able to see my current balance: so that I can manage my funds effectively and place bets.

### Icebox
- As a user, I want to be able to have random events make racers more variable: to add excitement and unpredictability to the races, making each event unique and engaging.
- As a user, I want weather effects to influence the race: to introduce additional challenges and such as rain affecting track conditions or wind affecting horse performance.
- As a user, I want to see cool stylized sprites: to enhance the visual appeal of the website and races.
- As a user, I want to hear crowd noises and see emojis: to add ambiance and atmosphere to the races, making them feel more realistic and lively.
- As a user, I want to be able to hear commentator messages: to provide commentary and insights during the races to enhance the entertainment value.
- As a user, I want to be able to see the amount of times a user has gone bankrupt: to track my own financial history and learn from past mistakes.
